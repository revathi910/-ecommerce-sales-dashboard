# 🛒 E-Commerce Sales Dashboard — Excel Project

## 📌 Overview
An interactive Excel dashboard analyzing sales performance 
of a global E-Commerce store across product categories, 
regions, and months.

Built as part of the **Business Analytics with Excel 
Certification Training** Course-End Project.

---

## 🎯 Objective
Design a dynamic sales dashboard that allows stakeholders 
to filter data by **Product Category** and view:
- Month-wise Sales & Profit trends
- Region-wise Sales breakdown

---

## 🗂️ Dataset
- **File:** E-Commerce Dashboard dataset.xlsx
- **Rows:** 51,290 sales records
- **Columns:** 21 features including Order ID, Sales, 
  Profit, Region, Category, Month, Customer info

### Product Categories:
- Auto & Accessories
- Electronic
- Home & Furniture
- Fashion

### Regions Covered:
- Oceania, Central, North, West, East, South, 
  Southeast Asia, Central Asia, EMEA, North Asia, 
  Africa, Caribbean, Canada

---

## 🛠️ Tools Used
- Microsoft Excel
- Data Analysis Add-in
- Form Controls (Combo Box)
- Excel Charts & Pivot Tables

---

## 📊 Dashboard Features
| Feature | Description |
|---|---|
| Combo Box | Filter by Product Category |
| Month-wise Chart | Sales & Profit trend per month |
| Region-wise Chart | Sales distribution by region |
| Dynamic Tables | Auto-update based on selection |

## 🚀 How to Use
1. Open `E-Commerce_Sales_Dashboard.xlsx`
2. Go to the **Dashboard** sheet
3. Use the **dropdown (Combo Box)** to select a category
4. Charts and tables update automatically
